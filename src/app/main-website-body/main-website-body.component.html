Você conhece Java?



Java é uma linguagem de programação e ambiente computacional criada pela empresa Sun Microsystems, na década de 90. Apesar de desenvolvida por esta, ela foi adquiroda pela Oracle.

Devido a sua capacidade de escrever um código apenas uma vez e rodá-lo em diversos dispositivos, Java tornou-se incrivelmente popular. A tecnologia foi implantada em diversos dispositivos, desde celulares até computadores.
